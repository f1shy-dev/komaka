# ink-v2

A project built with Bun, a fast all-in-one JavaScript runtime.

## Setup

Install dependencies:

```bash
bun install
```

## Usage

Run the project:

```bash
bun run index.ts
```

## Source Files

- `src/index.tsx`
- `src/types.d.ts`
- `src/aiRegistry.ts`
- `src/promptFramework.ts`

## Notes

This project was initialized with `bun init` in Bun v1.2.5.
